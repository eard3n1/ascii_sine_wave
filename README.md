<h1>ascii_sine_wave</h1>
<p></p>A simple and customizable terminal animation of a sine wave using C++ and Python.</p>

<h2>Necessities</h2>
<ul>
  <li>C++ compiler (g++, clang, or MSVC)</li>
  <li>Python 3+</li>
</ul>

<h2>Setup</h2>
<ul>
  <li>Compile <code>render.cpp</code> to an executable:
    <pre><code>g++ render.cpp -o render.exe</code></pre>
  </li>
  <li>Run the <code>main.py</code> file:
    <pre><code>python main.py</code></pre>
  </li>
</ul>

<h2>Note</h2>
<ul>
  <li>The executable will be ran inside <code>main.py</code>.</li>
  <li>There are a few options you can tweak inside <code>config.py</code></li>
</ul>
